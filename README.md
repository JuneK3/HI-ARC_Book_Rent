> **Intro**

- 공부하는데 있어서 책을 필요로 하는 학회원들에게 여러가지 이유로 책장에서 먼지가 쌓여가는 책들을 대여해주는 서비스(이사등의 이유로 양도하는 기능도 추가)

- Django LocalLibrary 프로젝트에 Rest API를 적용하여 CRUD를 오로지 Json을 리턴하는 API서버로서 개발
